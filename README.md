# Taptrap Deployment Guide

Es ist der offizielle Deployment Guide von Taptrap auf MacOS.
Es dient als Grundlage dafür wie man Taptrap auf einem Pixel 6a Emulator simulieren kann um Zugriffsrechte zu erhalten, ohne dass der Benutzer etwas von der Prozedur mitkriegt. Zusätzlich sollen Skripte bereitgestellt werden, welche die Analyzen bereitstellen.
